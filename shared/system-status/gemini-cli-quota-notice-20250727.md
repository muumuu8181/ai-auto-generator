## 📋 重要情報: Gemini CLIクォータ制限

### **発生事象**
**Gemini CLI日次クォータ制限**に到達した場合の動作変更について

### **従来の理解（誤り）**
```
❌ Pro制限 → 自動的にFlashに切り替わる
```

### **実際の動作（正確）**
```
✅ Pro制限 → Flash切り替え表示されるが実際には制限継続
✅ "Fallback to Flash model failed" エラーが発生
✅ 当日中はGemini CLI使用不可
```

### **対応方法**
1. **制限検知**: エラーメッセージで判断
2. **代替手段**: Claude Code標準機能で継続作業
3. **翌日復旧**: 日次リセットで通常利用再開
