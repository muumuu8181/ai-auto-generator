
#### 4. **曖昧ルール具体化システム**
```
機能:
✅ 曖昧な表現の自動検出

✅ 継続的ルール改善サイクル

- 「適切に」「十分に」等の主観的表現
- 「必要に応じて」等の判断基準不明表現  
- 手順・タイミングが不明確な指示
- AI個人判断に依存する部分
```
