## 📋 必須報告手順

### **Step 1: 作業開始時のバージョン確認**
作業開始時に以下のコマンドを実行・記録：

```bash
echo "=== 作業環境バージョン確認 ==="
cat VERSION.md | head -10
echo "現在のワーキングディレクトリ: $(pwd)"
git log --oneline -1
echo "作業開始時刻: $(date)"
```


## 🚫 報告不備への対応

### **バージョン情報未記載**
- ❌ **受付拒否**: 完了報告が受け付けられない
- ⚠️ **即座修正**: 24時間以内に追記必須
- 📝 **学習記録**: 「バージョン情報記載忘れ」をreflection.mdに記録

### **不正確な情報**

- 📊 **修正要求**: 正確な情報への即座修正
